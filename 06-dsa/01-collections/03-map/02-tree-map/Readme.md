### TreeMap

A TreeMap stores data in key-value pairs like HashMap, but the major difference is, TreeMap keeps keys automatically sorted.

### HashMap vs TreeMap

| Feature            | HashMap     | TreeMap         |
| ------------------ | ----------- | --------------- |
| Ordering           | No order    | Sorted order    |
| Performance        | Faster      | Slightly slower |
| Internal Structure | Hash Table  | Red-Black Tree  |
| Null Key           | One allowed | Not allowed     |
| Thread Safe        | No          | No              |
