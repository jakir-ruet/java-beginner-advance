### HashMap

A HashMap in Java stores data as key-value pairs. It is one of the most used collections in real-world backend applications such as:

- User session storage
- Configuration management
- Caching
- Inventory systems
- Employee records
- API response mapping
- Counting frequencies

| Feature            | HashMap     | TreeMap         |
| ------------------ | ----------- | --------------- |
| Ordering           | No order    | Sorted order    |
| Performance        | Faster      | Slightly slower |
| Internal Structure | Hash Table  | Red-Black Tree  |
| Null Key           | One allowed | Not allowed     |
| Thread Safe        | No          | No              |
